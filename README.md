## Dependencies

- [Yarn](https://classic.yarnpkg.com/en/docs/install/#mac-stable)

## Get Started

### Clone Repository

```
git clone https://github.com/wanmfsdev/cms-builder.git
```

### Run

```bash
yarn install
yarn dev
```
